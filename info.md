# OAuth Mail

[![hacs_badge](https://img.shields.io/badge/HACS-Custom-41BDF5.svg)](https://github.com/hacs/integration)

This integration provides OAuth authentication for email services and automatically configures the OAuth Mail Proxy addon with account details and tokens.

## Features

- OAuth 2.0 authentication for email providers
- Automatic proxy configuration
- Secure token encryption
- Support for Outlook and Gmail

## Installation

1. Add this repository to HACS as a custom repository
2. Install the "OAuth Mail" integration
3. Configure your email account through the integration setup

## Configuration

The integration handles all OAuth flows and proxy configuration automatically. Simply provide your email credentials and follow the authorization process.
